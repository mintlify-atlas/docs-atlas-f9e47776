Awesome Hacker Search Engines is a community-maintained reference of the best security-focused search engines and online tools. Whether you're enumerating attack surfaces, hunting CVEs, tracking threat actors, or running OSINT investigations, this list has you covered — organized into 25+ categories so you can find the right tool fast.

## About This Project

Awesome Hacker Search Engines is an open-source, MIT-licensed project maintained by [edoardottt](https://edoardottt.com/) and the community. All tools listed are publicly accessible online resources — no software installation required.

<Note>
  This list is for educational and authorized security testing purposes only. Always ensure you have proper permission before using these tools against any target.
